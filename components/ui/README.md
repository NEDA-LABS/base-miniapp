# UI Components

This directory contains shadcn/ui components for the application.

## Structure

All shadcn/ui components should be placed in this directory following the standard shadcn/ui conventions.

## Usage

Components can be added using the shadcn/ui CLI:

```bash
npx shadcn-ui@latest add [component-name]
```

## Available Components

Components will be listed here as they are added to the project.
